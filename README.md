# Customer Segmentation using K-Means clustering 
# Objective:
 To perform customer segmentation on retail/mall-style customer data using K-Means clustering in python.
## Key Insights section
Identified High-Income, High-spending customers as the core target for premium offers and loyalty programs.
Found High-Income, Low-spending customers who may need better engagement or upselling.
Spotted Low-Income, High-spending customers as high-risk, needing careful credit and promotional polices.
Used cluster labels as input for future supervised tasks.
